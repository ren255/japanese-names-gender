# 日本人名前データ

4つのデータセットから作成されています。

| dataset     | size | 割合 | 男性割合 | ソース                  | note                                 |
| ----------- | ---- | ---- | -------- | ----------------------- | ------------------------------------ |
| facebook    | 150k | 36%  | 57.2%    | facebook5億人漏洩データ | 漢字またはひらがなのどちらかのみ     |
| enamdict    | 116k | 28%  | 16.4%    | 固有名詞辞書            | 分布に大きな特徴あり                 |
| name_origin | 90k  | 21%  | 60.4%    | 名前由来                | 性別ごとの漢字とその読みデータの展開 |
| gendec      | 64k  | 15%  | 49.8%    | 一般的な名前データ      |                                      |
